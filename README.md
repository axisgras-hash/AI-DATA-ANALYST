# 🤖 Auto EDA using AI

An intelligent **Automated Exploratory Data Analysis (EDA)** web application powered by **AI (Google Gemini)** and built with **Streamlit**.  

This tool allows users to upload datasets and instantly generate:
- 📊 Data insights  
- 📈 Visualizations  
- 🧠 AI-powered summaries  
- 💬 Natural language queries on data  

---

## 🚀 Features

### 📌 1. Automated EDA
- Dataset overview (shape, columns)
- Missing value analysis
- Data cleaning preview
- Statistical summaries (numerical + categorical)

### 📊 2. Visualizations
- Correlation heatmap
- Pair plots
- Distribution plots
- Categorical analysis charts

### 🧠 3. AI-Powered Insights
- Generate dataset summary using **Gemini AI**
- Understand dataset context & domain automatically

### 💬 4. Chat with Your Data
- Ask questions in natural language
- Example:
  - “What is the average sales?”
  - “Show distribution of age”
  - “Are there missing values?”

### ⚡ 5. AI Code Generation Engine
- Converts user queries → executable Python code
- Automatically runs analysis using:
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn

### 📄 6. AI Report Generation
- Generate complete analytical reports using AI

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **AI Model:** Google Gemini (`gemini-2.5-flash`)  

**Libraries:**
- pandas
- numpy
- matplotlib
- seaborn
